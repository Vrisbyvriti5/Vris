## Included Projects

- `nirvi-elevated-style/` — the main React + Vite storefront application
- `nirvi-Backend-style/` — Node.js + Express backend API

## Getting Started

### Frontend
```bash
cd nirvi-elevated-style
npm install
npm run dev
```

### Backend
```bash
cd nirvi-Backend-style
npm install
# Configure your .env file first
npm run dev
```

## Setup Notes

1. Copy `.env` template in the backend and fill in your credentials
2. Run `node config/initDb.js` to initialize the database
3. Run `node config/seed.js` to seed sample data (optional)
4. Start both frontend and backend dev servers
